# Archived ShipFree demo landing

These are the original ShipFree boilerplate landing-page components, archived
when the Spruce Ridge Wellness marketing site was migrated in.

They are excluded from TypeScript compilation (see `tsconfig.json`) and the
folder is prefixed with `_` so Next.js will not pick anything up as a route.

Kept for reference / future use — e.g., if a pricing or feature page is needed
for the dashboard SaaS phase later.
